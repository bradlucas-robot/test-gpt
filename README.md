# 星际小宇航员 AI 学园

一个面向有宇航员梦想的小学生的互动式前端工程，融合了：

- 天文知识卡片
- 小学生友好的问答挑战
- AI 航天伙伴对话模块
- 宇航员任务训练舱
- 成长积分与身份激励

## 设计目标

这个项目不是普通的静态页面，而是一个具备产品原型思路的儿童航天教育前端：

1. **激发梦想**：通过角色身份、任务闯关和视觉氛围，让孩子感受到“我正在成为宇航员”。
2. **普及知识**：以太阳、月球、火星、空间站等低门槛知识点作为切入。
3. **融入 AI**：设置儿童友好的 AI 航天伙伴“星航”，提升陪伴感与互动感。
4. **便于扩展**：项目按模块拆分，后续可以接入真实 AI API、增加更多小游戏、加入家长端或课堂端。

## 技术栈

- React
- Vite
- 原生 CSS

## 本地运行

```bash
npm install
npm run dev
```

## 当前功能结构

```text
src/
  App.jsx       # 主界面与核心交互逻辑
  data.js       # 知识卡片、题库、任务与 AI 文案数据

```
